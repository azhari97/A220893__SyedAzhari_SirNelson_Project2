Goal:

Develop a campus marketplace application that reduces waste by encouraging students and staff to resell, exchange, and donate usable items within the university community.

Lab Submissions

Lab 1: UI Design

Focus on layout structure, alignment, and basic composables in Jetpack Compose.

This lab improved my understanding of the fundamentals of user interface design for mobile applications. I learned how layout organization, color selection, and component positioning influence the overall user experience. It helped me develop better practices for creating simple, organized, and user-friendly interfaces.

Lab 2: Interaction

Handling screen state and dynamic user inputs.

Through this lab, I gained a better understanding of handling user interactions, including button actions and user input. I learned how applications can respond to user behavior and update dynamically. This experience helped me create apps that are more responsive and functional.

Lab 3: Material 3 Design

Implementing Material 3 components for a refined user interface.

This lab introduced me to the Material 3 Design system for mobile application development. I learned how to implement UI components such as cards, buttons, and consistent color schemes to create a more polished interface. It improved my ability to design applications with better visual consistency and user experience.

Lab 4: Navigation

Setting up a navigation for the mobile app to manage screen transitions and routing.

In this lab, I learned how to implement screen transitions using Jetpack Navigation components. I understood how organizing different screens and routes can make an application easier to navigate. This helped improve the overall structure and user flow of my mobile application.

Lab 5: Room Database

Persistent local storage using NoSQL-based Room library.

This lab provided me with experience in managing local data storage using the Room Database. I learned how to store, modify, and retrieve information within an application. It improved my understanding of implementing offline data handling in Android applications.

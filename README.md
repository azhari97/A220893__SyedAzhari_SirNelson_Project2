SDG THEME

SDG 12 — Responsible Consumption and Production

Problem Focus:
Reducing unnecessary waste within university communities.
Encouraging students and staff to reuse, resell, exchange, or donate usable items instead of throwing them away.
Promoting a sustainable campus marketplace culture.

CORE FEATURES:
User Management
User registration and login simulation using email validation.
Local user persistence using Room Database.
Edit and update user profile information.
Logout and clear local user data.
Campus Marketplace
Browse available products from other users.
Search and filter products by category/type.
View detailed product information.
Add new products for resale or donation.
Upload product images using camera sensor.
Display product images from online storage.
Cart & Transaction System
Add products into cart.
Separate cart status:
In Cart
Pending Offer
Accepted
Rejected
Completed
Update transaction details such as:
Offer price
Meeting location
Additional details
Store and sync transaction data using Firebase Firestore.
SDG News Integration
Fetch live SDG 12 related news from external API.
Display dynamic environmental and sustainability updates.
Data Management
Offline storage using Room Database.
Cloud storage using Firebase Firestore.
State management using ViewModel.
